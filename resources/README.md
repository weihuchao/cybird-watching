# 小鸟配置示例

## 配置文件格式

小鸟配置使用JSON格式，包含以下字段：
- `id`: 小鸟的唯一标识符（数字）
- `name`: 小鸟的显示名称（中文）
- `weight`: 权重值，影响随机选择的概率（数字，越大越容易被选中）

## 资源目录结构

```
/birds/
├── 1001/          # 小鸟ID
│   ├── 1.bin      # 第1帧动画
│   ├── 2.bin      # 第2帧动画
│   ├── 3.bin      # 第3帧动画
│   └── ...        # 更多帧...
├── 1002/
│   ├── 1.bin
│   ├── 2.bin
│   └── ...
└── ...
```

## 配置说明

1. **ID与目录对应**: 配置中的`id`字段必须与资源目录名完全一致
2. **图片编号**: 从1开始连续编号，格式为`{帧号}.bin`
3. **任意帧数**: 支持任意数量的帧数，系统会自动检测
4. **全局帧率**: 固定8fps（125ms每帧），后续可配置调整

## 权重系统

权重影响小鸟被随机选中的概率：
- 总权重 = 所有小鸟权重之和
- 每只鸟被选中概率 = 该鸟权重 / 总权重
- 例如：麻雀(50) + 知更鸟(30) = 总权重80，麻雀选中概率为50/80=62.5%

## 使用方法

1. 将`bird_config.json`复制到SD卡的`/configs/`目录下
2. 创建对应的资源目录结构
3. 将RGB565格式的图片文件（.bin）放入对应目录
4. 系统启动后会自动加载配置并开始工作
